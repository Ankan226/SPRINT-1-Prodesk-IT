# Prodesk IT Landing Page

## Overview

A modern, responsive landing page developed for the Prodesk IT Sprint 01 Internship Assignment.

The project was built in three phases:

* Phase 1: Base MVP using HTML5 and CSS3
* Phase 2: JavaScript enhancements including Dark/Light Mode and mobile navigation
* Phase 3: Tailwind CSS refactor with performance and SEO optimizations

---

## Features

### Phase 1

* Responsive Navbar
* Hero Section
* Services Section
* Footer with Social Links
* Responsive Layout using Flexbox/Grid

### Phase 2

* Dark/Light Theme Toggle
* Theme Persistence using localStorage
* Sticky Navbar
* Hover Animations
* Mobile Hamburger Menu

### Phase 3

* Tailwind CSS Refactor
* Glassmorphism Navbar
* SEO Optimization
* Accessibility Improvements
* Lighthouse Optimization
